How to setup a free website with cloudflare and github

How to make a webpage
  • Go to github.com
  • Then, sign up for github
  • Click create a repository
  • Now, make a new file
  • Name your file but at the end put .html. For example, test.html
  • Type whatever html code you want or click

    • here

    You may also add as many files as you like.

  • Next, Click "commit changes"
  • Then, go to cloudflare.com and press start for free
  • Next, click "Add a website" and make an account
  • Now, click workers and pages
  • Select create application
  • Then, click on pages then, connect to git
  • Next, select Add account then sign in to your github account after you do that you select your repository
  • Now, click begin startup
  • Next, scroll down and click save and deploy
  • wait till it's done and press continue to project at the bottom
  • You're all done! Click Visit site to see your site.
