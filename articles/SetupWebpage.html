How to setup a free website with cloudflare and github

How to make a webpage
  • Go to github.com
  • Then, sign up for github
  • Click create a repository
  • Now, make a new file
  • Name your file but at the end put .html. For example, test.html
  • Type whatever html code you want or click

    • here

    You may also add as many files as you like.

  • Next, Click "commit changes"
  • Then, go to cloudflare.com and press start for free
  • Next, click "Add a website" and make an account
